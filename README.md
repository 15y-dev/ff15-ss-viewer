# FF15 スナップショットビューア

FINAL FANTASY XV の `.ss` スナップショットファイルをブラウザで閲覧・保存できるツールです。
サーバーへのデータ送信は一切なく、すべての処理はブラウザ内で完結します。

## 使い方

1. `https://15y-dev.github.io/ff15-ss-viewer/` をブラウザで開く
2. ファイル選択エリアをクリックして `.ss` ファイルを選択（複数可）、またはファイルをドラッグ＆ドロップ
3. サムネイル一覧が表示されます

### ライトボックス（拡大表示）

- サムネイルをクリック → 拡大表示
- `←` / `→` キーまたはボタンで前後に移動
- `Escape` キーまたは×ボタンで閉じる

### JPEGとして保存

- サムネイルにカーソルを合わせると表示されるダウンロードボタンをクリック
- ライトボックス内の「JPEGで保存」ボタンをクリック

ファイル名は元の `.ss` ファイル名の拡張子を `.jpg` に変えたものになります。

## SSファイルの場所

```
C:\Users\<ユーザー名>\Documents\My Games\FINAL FANTASY XV\Steam\<SteamID>\savestorage\snapshot\
```

## 技術的な仕組み

`.ss` ファイルはゲーム独自のヘッダーが先頭に付いたJPEGファイルです。
このツールはファイルのバイナリを走査してJPEGヘッダー（`FF D8 FF`）を探し、
それ以前のゲーム固有データを除去することで画像として表示します。

## 動作環境

モダンブラウザ（Chrome / Edge / Firefox / Safari）であれば動作します。
インストール不要、インターネット接続不要です。

## プライバシー

画像データを含むすべての処理はブラウザ内のみで行われます。
外部サーバーへの通信は一切発生しません。
